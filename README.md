# ASK AI

ASK AI is a web-based chatbot that supports real-time AI conversations and PDF uploads for summaries. It features multiple chats, dark mode, and a clean interface.

---

## Features

- Chat with AI in real-time.
- Upload PDFs for summaries.
- Multiple chat sessions (rename/delete).
- Dark mode toggle.
- Modern, responsive design.

---

## Installation

```bash
git clone https://github.com/Ismailk12/ASK-AI.git
cd ASK-AI
python3 -m venv venv
source venv/bin/activate   # Linux/macOS

pip install -r requirements.txt
